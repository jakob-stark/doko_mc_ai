Detailed Description

A random number generator module

Both functions are based on the same xorshift prg by George Marsaglia. A detailed description may be found at Wikipedia. Here, the basic 32bit variant is used, which should suffice for our mc sampling.

Note
Implementation detail: the following transforms the 32bit random number in a IEEE754 single precision floating point number between 1 and 2 by setting setting the sign bit (msb) to zero and the exponent (8 bits following the sign bit) 
seee eeee emmm mmmm mmmm mmmm mmmm mmmm
|\--------/\--------------------------/
0   127         randomly filled
the floating point number is 
(-1)^s * 2^((e)_2 - 127) * (1.m)_2
 = (-1)^0 * 2^0 * (1.m)_2
 = (1.m)_2
