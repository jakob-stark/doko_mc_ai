Samples card distribution.

This function samples a card distribution according to the information given in card_info. It doesn't preserve dest nor card_info, so make sure to copy them.

Parameters
destDestination GameInfo object. Will only add cards to the cardsets of players 1-3. So this can be used with an initialized GameInfo object where the remaining card will be filled in.
card_infoInformation about which cards are to be dealed and with which scores.
random_statepointer to 32bit value used as random state by Random and RandomInt

int sort_and_check ( CardInfo *  card_info)

prepares the card_info structure

This function checks if card_info is valid and consitent. If it is, the cards are sorted ascending by their contraints, so that the mc sampler deals the cards with most contraints first.

Parameters
card_infoStructure to operate on
Returns
0 on sucesss, a nonzero value if the card_info struct is inconsistent
