◆ Simulate()

Score Simulate ( const GameInfo *  game_info,
CardId  next_card,
uint32_t *  random_state 
)

Simulates a random game.

First play the card denoted by next_card onto the game in game_infe. After that, random but legal cards are played until the game is over. The score that was achieved by player0's team is returned.

Note
The first card is given in next_random instead of being random. This allows the core functions to simulate a random game with a specific card and study how well that card behaves.
Parameters
game_infopointer to GameInfo struct to simulate
next_cardthe first card, that is to be played
random_statepointer to 32bit random state used by Random and RandomInt
Returns
Score value for player 0
